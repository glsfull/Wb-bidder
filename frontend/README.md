# WB Bidder frontend

This folder contains the Nuxt 3 frontend workspace for WB Bidder.

## Current scope

The first implementation phase defines the application shell and product direction. The frontend is prepared for:

- Public landing pages.
- Authentication and store onboarding.
- Private dashboard shell.
- Main bidder workspace.
- Analytics, AI advisor, reports, and settings sections.

## Planned commands

After dependencies are installed:

```bash
npm install
npm run dev
npm run build
npm run lint
npm run test
```

## Structure

```text
app/          Nuxt application entry and pages
components/   Shared UI components
public/       Static public assets
server/       Nuxt server routes and server utilities
tests/        Frontend tests
```
